Lista del proceso

  1. Usé el elemento "figure" para para introducir una imagen y un titulo que tuviera sentido con el tema de la pagina.
  2. Creé un footer, despues, dentro de este usé el elemento "picture" para introducir varias imagenes que cambiaran dependiendo del tamaño de la ventana del navegador.
  3. Entre el footer y el formulario, usé el elemento video para insertar un video del instituto con distintos formatos que se inician solo.
  4. Despues del video, usé el elemento "audio" para insertar un audio de musica en distintos formatos, el cual le puse los controles para poder pararlo e iniciarlo.
  5. Por ultimo, hice esta lista ordenada con el elemento "ol".
